# SwarmEx

SwarmEx is an autonomous multi-agent DeFi execution engine.

Setup:
1. `npm install`
2. Follow KeeperHub manual setup.
3. Faucets: Base Sepolia (`https://www.alchemy.com/faucets/base-sepolia`), 0G Testnet (`https://faucet.0g.ai`), USDC (`https://faucet.circle.com`).
4. Set `.env` values based on `.env.example`.
